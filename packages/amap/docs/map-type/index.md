## API

| 属性 |说明|类型|默认值|amap版本|
|-----|----|----|----|----|
|visiable| 是否显示 | `boolean` | `true` | -- |
|defaultType| 初始化默认图层类型。取值为0：默认底图 取值为1：卫星图 默认值：0 | `0` \| `1` | `0` | -- |
|showTraffic| 是叠加实时交通图层 | `boolean` | `false` | -- |
|showRoad| 叠加路网图层 | `boolean` | `false` | -- |
